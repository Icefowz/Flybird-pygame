# Flybird pygame

A simple Flappy Bird-style game built with Python and Pygame.

## Technologies

- Python
- Pygame
- Standard library modules: `random`

## Project Overview

This repository contains a small arcade game where the player controls a bird passing through moving pipes. The game uses Pygame for rendering, input handling, and animation.

## Requirements

- Python 3.8+
- Pygame

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Icefowz/Flybird-pygame.git
   cd Flybird-pygame
   ```

2. Install Pygame:

   ```bash
   pip install pygame
   ```

## Running the Game

Run the main game file:

```bash
python flybird.py
```

## Controls

- Press `SPACE` to make the bird jump.
- Close the game window or press the close button to exit.

## Notes

- The current code loads image assets from absolute file paths. To run the game correctly, you should place the required image files in the project directory and update the paths in `flybird.py` accordingly.
